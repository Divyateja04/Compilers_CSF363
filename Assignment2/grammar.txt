S->PROGRAMSTRUCT VARIABLES PROGRAMBLOCK
PROGRAMSTRUCT ->PROGRAM PROGRAMNAME SEMICOLON NL
PROGRAM -   
SEMICOLON - ;
FULLSTOP - .
COLON - :
EQUALS - =
RELOP - EQUAL | LESS |  GREATER | LESSEQUAL | GREATEREQUAL | NOTEQUAL
VARIABLES ->var NL VARS NL
VARS ->VARIABLELIST COLON TYPE SEMICOLON
VARIABLELIST ->VARIABLENAME COMMA VARIABLELIST | VARIABLENAME
OPERATOR
PROGRAMBLOCK ->BEGIN BODY END FULLSTOP
BODY -> STATEMENTLIST
STATEMENTLIST ->READ_STATEMENT | WRITE_STATEMENT | ASSIGNMENT_STATEMENT | BLOCK_STATEMENT | CONDITIONAL_STATEMENT | LOOPING_STATEMENT
READ_STATEMENT ->READ LPAREN VARIABLENAME RPAREN SEMICOLON NL
READ - read
WRITE_STATEMENT ->WRITE LPAREN VARIABLENAME RPAREN SEMICOLON NL | WRITE LPAREN DOUBLEQUOTES STRING DOUBLEQUOTES RPAREN SEMICOLON NL
WRITE - write
ASSIGNMENT_STATEMENT -> VARIABLENAME COLON EQUAL EXPRESSION SEMICOLON NL
EXPRESSIONLIST -> EXPRESSION EXPRESSIONLIST | EXPRESSION
EXPRESSION -> IDENTIFIER OPERATOR IDENTIFIER | IDENTIFIER OPERATOR NUMBER | IDENTIFIER OPERATOR 
BLOCK_STATEMENT -> BEGIN BODY END SEMICOLON
CONDITIONAL_STATEMENT -> 
CONDITION -> IDENTIFIER RELOP IDENTIFIER | IDENTIFIER RELOP IDENTIFIER
BLOCK -> BEGIN NL EXPRESSIONLIST NL END SEMICOLON NL
LOOPING_STATEMENT -> FOR IDENTIFIER COLON EQUALS NUMBER TO EXPRESSION DO NL | FOR IDENTIFIER COLON EQUALS NUMBER DOWNTO EXPRESSION DO NL | WHILE LPAREN CONDITION RPAREN DO NL BLOCK
