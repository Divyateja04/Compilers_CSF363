S->PROGRAMSTRUCT VARIABLES PROGRAMBLOCK
PROGRAMSTRUCT ->PROGRAM PROGRAMNAME SEMICOLON NL
VARIABLES ->VAR NL VARS NL
VARS -> MULTIVAR VARS | EPSILON
MULTIVAR -> VARIABLELIST COLON TYPE SEMICOLON NL | ARRAYDEF
VARIABLELIST ->VARIABLENAME COMMA VARIABLELIST | VARIABLENAME

PROGRAMBLOCK ->BEGINK BODY END DOT
BODY -> STATEMENTLIST BODY | EPSILON
STATEMENTLIST ->READ_STATEMENT | WRITE_STATEMENT | ASSIGNMENT_STATEMENT | BLOCK_STATEMENT | CONDITIONAL_STATEMENT | LOOPING_STATEMENT

READ_STATEMENT ->READ LPAREN VARIABLENAME RPAREN SEMICOLON NL

WRITE_STATEMENT ->WRITE LPAREN VARIABLENAME RPAREN SEMICOLON NL | WRITE LPAREN DQUOTE STRING DQUOTE RPAREN SEMICOLON NL

ASSIGNMENT_STATEMENT -> VARIABLENAME COLON EQUAL EXPRESSION SEMICOLON NL
EXPRESSION -> EXPRESSIONID | EXPRESSIONNUM | EXPRESSIONOP
EXPRESSIONOP -> IDENTIFIER OPERATOR IDENTIFIER | IDENTIFIER OPERATOR NUMBER | NUMBER OPERATOR IDENTIFIER 
EXPRESSIONNUM -> NUMBER
EXPRESSIONID -> IDENTIFIER

BLOCK_STATEMENT -> BEGINK BODY END SEMICOLON
CONDITION -> IDENTIFIER RELOP IDENTIFIER | IDENTIFIER RELOP IDENTIFIER
RELOP - EQUAL | LESS |  GREATER | LESSEQUAL | GREATEREQUAL | NOTEQUAL
CONDITIONAL_STATEMENT -> SIMPLEIF | IFTHENELSE
SIMPLEIF -> IF CONDITION THEN BLOCK_STATEMENT
IFTHENELSE -> IF CONDITION THEN BLOCK_STATEMENT ELSE BLOCK_STATEMENT

LOOPING_STATEMENT -> FORTO | FORDOWNTO | WHILEDO
FORTO -> FOR IDENTIFIER COLON EQUALS NUMBER TO EXPRESSIONOP DO BLOCK_STATEMENT | FOR IDENTIFIER COLON EQUALS NUMBER TO EXPRESSIONID DO BLOCK_STATEMENT | FOR IDENTIFIER COLON EQUALS NUMBER TO EXPRESSIONNUM DO BLOCK_STATEMENT 
FORDOWNTO -> FOR IDENTIFIER COLON EQUALS NUMBER DOWNTO EXPRESSIONOP DO BLOCK_STATEMENT | FOR IDENTIFIER COLON EQUALS NUMBER DOWNTO EXPRESSIONID DO BLOCK_STATEMENT | FOR IDENTIFIER COLON EQUALS NUMBER DOWNTO EXPRESSIONNUM DO BLOCK_STATEMENT 
WHILEDO -> WHILE LPAREN CONDITION RPAREN DO NL BLOCK_STATEMENT

DATATYPE -> INTEGER | REAL | BOOLEAN | CHAR 
ARRAYDEF -> IDENTIFIER COLON ARRAY LBRACKET NUMBER DOT DOT NUMBER RBRACKET OF DATATYPE SEMICOLON
